I think normal portfolios are kinda boring. I wanted to build something that feels like J.A.R.V.I.S or a real hacker terminal. So I made this system archive. It has all my projects, audio labs stuff, and even some playable games I coded.

I didnt use any heavy frameworks or libraries like React. Just raw vanilla code.

// COOL STUFF INSIDE
J.A.R.V.I.S Theme: It has a deep blue holographic look with crt monitor lines and a custom target cursor.

Omega Override: Try pressing Shift + K anywhere on the screen. The whole system locks down and flips into a dark matrix hacker mode.

Real Sound System: There are no .mp3 files here. I build a custom audio engine using Web Audio API to make all the system beeps in real time.

Playable Wargames: You can actually play games in the terminal menus. I added Snake (Ouroboros), a reflex test, Rock Paper Scissors, and Tic-Tac-Toe.
